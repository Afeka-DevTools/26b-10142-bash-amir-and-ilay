## חלוקת הסקריפטים בצוות
* **הסקריפטים שנכתבו ע"י אילי:**
  1. גיבוי תכולה של תיקיה לקובץ `tar.gz` (`backup_dir.sh`)
  2. שליפת מזג האוויר המעודכן לעיר מסוימת (`get_weather.sh`)
  3. מחיקת קבצים הישנים יותר מ-X ימים (`delete_old_files.sh`)
  4. סיכום מספר הקבצים, התיקיות והלינקים בתיקייה (`dir_summary.sh`)
  5. ניקוי קבצים זמניים מפרויקט (`clean_project.sh`)
     הוראות הפעלה וקלטים עבור כל סקריפט

* **הסקריפטים שנכתבו ע"י אמיר קשאני:**
1. בדיקת חיבוריות לרשת האינטרנט והדפסת log למסך
2.      
א. גיבוי תיקייה לקובץ tar.gz
קלט נדרש: נתיב לתיקייה שברצונכם לגבות כארגומנט.

פקודת הרצה:


./scripts/backup_dir.sh /path/to/folder
ב. שליפת מזג אוויר מעודכן
תיאור: הסקריפט בודק אם חבילת curl מותקנת, ובמידה ולא - יתקין אותה אוטומטית (מצריך הרשאות sudo). ניתן להעביר עיר בארגומנט או להזין אותה בצורה אינטראקטיבית.

פקודת הרצה:


./scripts/get_weather.sh London
ג. מחיקת קבצים ישנים יותר מ-X ימים
קלט נדרש: נתיב לתיקייה היעד ומספר הימים המקסימלי לשמירה.

פקודת הרצה:

./scripts/delete_old_files.sh /path/to/folder 7
ד. סיכום תכולת תיקייה
קלט נדרש: נתיב לתיקייה (אופציונלי, במידה ולא יסופק הסקריפט ינתח את התיקייה הנוכחית).

פקודת הרצה:

./scripts/dir_summary.sh /path/to/folder
ה. ניקוי קבצים זמניים מפרויקט
תיאור: סורק ומנקה קבצי .class, .pyc ותיקיות node_modules מהמיקום הנוכחי ומטה.

פקודת הרצה:


./scripts/clean_project.sh
