## מדריך הרצה ושימוש בפרויקט

### 1. שכפול המאגר המשותף (Clone)
פתחו טרמינל (Git Bash) והריצו:
```bash
git clone https://github.com/Afeka-DevTools/26b-10142-bash-amir-and-ilay.git
cd 26b-10142-bash-amir-and-ilay
```

### 2. הענקת הרשאות הרצה לכל הסקריפטים
```bash
chmod +x scripts/*.sh
```

### 3. הרצת כל סקריפט

#### `backup_dir.sh` — גיבוי תיקייה
**קלט נדרש:** נתיב לתיקייה שברצונכם לגבות (כארגומנט).
```bash
./scripts/backup_dir.sh /path/to/folder
```

#### `get_weather.sh` — שליפת מזג אוויר מעודכן
**תיאור:** בודק אם `curl` מותקן, ומתקין אוטומטית אם לא (מצריך הרשאות sudo). ניתן להעביר עיר כארגומנט או להזין אותה בצורה אינטראקטיבית.
```bash
./scripts/get_weather.sh London
```

#### `delete_old_files.sh` — מחיקת קבצים ישנים
**קלט נדרש:** נתיב לתיקיית היעד, ומספר הימים המקסימלי לשמירה.
```bash
./scripts/delete_old_files.sh /path/to/folder 7
```

#### `dir_summary.sh` — סיכום תכולת תיקייה
**קלט נדרש:** נתיב לתיקייה (אופציונלי — ברירת מחדל: התיקייה הנוכחית).
```bash
./scripts/dir_summary.sh /path/to/folder
```

#### `clean_project.sh` — ניקוי קבצים זמניים מפרויקט
**תיאור:** סורק ומנקה קבצי `.class`, `.pyc` ותיקיות `node_modules` מהמיקום הנוכחי ומטה.
```bash
./scripts/clean_project.sh
```

#### `check_internet.sh` — בדיקת חיבוריות לאינטרנט
**תיאור:** בודק חיבור לאינטרנט ומדפיס תוצאה + כותב ללוג (`connectivity_log.txt`).
**קלט נדרש:** אין
```bash
./scripts/check_internet.sh
```

#### `port_scanner.sh` — סריקת פורטים פתוחים
**תיאור:** בודק אם `nmap` (או `nc`) מותקן, ומתקין אוטומטית אם לא. סורק טווח פורטים נפוץ בכתובת IP נתונה ומדפיס אילו פורטים פתוחים.
**קלט נדרש:** כתובת IP (וטווח פורטים אופציונלי).
```bash
./scripts/port_scanner.sh 192.168.1.1
```

#### `generate_password.sh` — יצירת סיסמה רנדומלית
**תיאור:** יוצר סיסמה בת 10 תווים המכילה לפחות אות גדולה, אות קטנה, ספרה וסימן אחד.
**קלט נדרש:** אין
```bash
./scripts/generate_password.sh
```

#### `count_by_extension.sh` — ספירת קבצים לפי סיומת
**תיאור:** סורק תיקייה נתונה ומציג טבלת סיכום — כמות קבצים לכל סיומת.
**קלט נדרש:** נתיב לתיקייה (אופציונלי — ברירת מחדל: התיקייה הנוכחית).
```bash
./scripts/count_by_extension.sh /path/to/folder
```

#### `countdown_timer.sh` — טיימר ספירה לאחור
**תיאור:** מקבל משך זמן בפורמט HH:MM:SS ומציג ספירה לאחור חיה עד לסיום.
**קלט נדרש:** משך זמן בפורמט HH:MM:SS.
```bash
./scripts/countdown_timer.sh 00:05:00
```
